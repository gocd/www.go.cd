Get Started with GoCD in 2 Minutes

  • Step 1:

    Download the Dependencies

    All you need is Git, Docker, Docker Compose, and ensure Docker is running.

  • Step 2:

    Run the Commands

    Visit the Github repo and run the commands to get started.

    Link: https://github.com/arvindsv/gocd-starter-template-test

  • Step 3:

    Explore GoCD

    Try out a few things to get familiar with GoCD:

    • Create a pipeline
    • Add a stage
    • Configure a plugin
    • Edit the configuration
    • Explore more ways to use GoCD
